Top View:

Top View provides a high level graphical presentation of the diagram and allows easy navigation through clicking on each graphic.

You can access information about each graphical object in the diagram by clicking on it. Once you click on the object, MAP will open a page that displays:

  • A full description of the tool.
  • A list of all diagrams the tool is included in and allows access to them.
  • A list of all drilldown diagrams that the tool is included in and allows access to them.
  • A list of other related diagrams to the tool and allows access to them.
  • A list of all linked files and allows access to them.

  • Click on the (+) sign at the bottom of a graphical object to view a graphic presentation of the anchored diagram. If the tool is assigned to more than one anchored diagram, you’ll see a list with all diagrams from which you can select the one you want.

  • In case of an anchored diagram, click the (+) sign to view the needed information.
